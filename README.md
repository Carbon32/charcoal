## About:

<img src = "https://i.imgur.com/4mfc800.png" width = 100% height = 100%>


This is a Python Code Editor, made in Python. The editor supports multiple features including:

* Opening & closing folders/files.
* Syntax highlighting.
* Auto-completion.
* Advanced searching system.

The editor is still being developed, more features will be available soon enough.

## Installation:

To be able to run Charcoal, you need to have some libraries installed on your computer.

<b>NOTE:</b> Messing with directories may cause some unforeseen consequences...

1. Clone the repository: 

   ```sh
   git clone https://github.com/Carbon32/charcoal.git
   ```
2. Install all necessary libraries:

    ```sh
    pip install pyqt5
    pip install jedi
    ```
